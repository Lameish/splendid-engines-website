# LinkedIn brand assets

Source files for Splendid Engines LinkedIn imagery. Built against the locked visual brand in `our-brand/splendid-visual-brand/SKILL.md`.

## Files

### `banner-people-follow-people.svg`

LinkedIn personal banner, 1584 x 396 (4:1).

Design notes:

- Lagoon (`#153047`) background, per the social-imagery rule in section 9.
- Headline "PEOPLE FOLLOW PEOPLE." in white Hanley uppercase, with the closing "PEOPLE." in Poppy (`#DB4937`) for single-word emphasis. Daffodil is intentionally not used here: the highlighter rule (section 5) is light-surface only, so Poppy carries the accent role on dark.
- Sociogram on the right side: five-node hand-drawn network with one Poppy "buyer" node. Sits to the right so the LinkedIn profile pic (which overlaps the banner's bottom-left on desktop) does not cover it. This is the one "wink" on the asset.
- Small "SPLENDID ENGINES" wordmark in the bottom-right corner as a placeholder. Production should swap this for the white-on-Lagoon stamp logo PNG from Brandon.

Font substitutions in the SVG: Big Shoulders Display in place of Hanley Pro Slim Bold. The substitute is for layout review only. Final production should re-render with the licensed Hanley face before publishing.

## Exporting

To produce the PNG/JPG that LinkedIn accepts:

1. Open the SVG in a browser or vector tool (Figma, Illustrator, Inkscape).
2. Export at 1584 x 396 (1x) or 3168 x 792 (2x for retina).
3. Save as PNG or JPG under 8 MB.

Upload via LinkedIn profile -> background photo.
