Usage Rules

Color. Icons inherit from currentColor. Default to Lagoon on light surfaces, white on dark. Poppy is allowed when the icon is acting as a methodology accent (rare). Never Daffodil, never Lake as the primary icon color.

Size. Web inline 24px, web section markers 48px. Decks: 32-40px next to eyebrows, 72-96px in 3-up grids, 200px+ as hero. Slides require SVG (never PNG) for projection clarity.

Don't. No rotation, no skew, no recoloring outside the palette, no rounded-corner variants. If an icon needs adaptation, redraw it inside the system, don't filter the SVG.

Naming. Files use kebab-case matching the concept, e.g. content-program.svg, arrow-right.svg. The library lives at icons/v1/ in production.
